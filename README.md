# Ejecutar en desarrollo

1. Clonar el repo
2. Ejecutar
```
npm i
```

3. Tener Nest Cli instalado
```
npm i -g @nestjs/cli
```

4. Levantar la base de datos
```
docker-compose up -d
```

5. Cargar datos
```
http://localhost:3000/api/v2/seed
```

## Stack usado
* MongoDB
* Nest
